# VYRA — The Social Universe of 2125

Futuristic social platform: Flutter client, Node/TypeScript API, PostgreSQL, Socket.IO, OpenAI VYRA AI.

**Tagline:** The Social Universe of 2125 — not a clone of legacy social apps.

## Structure

| Path | Purpose |
|------|---------|
| `client/` | Flutter app (Riverpod, GoRouter, 2125 design system) |
| `server/` | Express API + Prisma + Socket.IO + VYRA AI |
| `database/` | Schema notes (source of truth: `server/prisma`) |
| `shared/` | TypeScript API contracts |
| `docs/` | INSTALL, DEPLOY, APK_BUILD |
| `deployment/` | Docker, Nginx, PM2, Railway, Render |
